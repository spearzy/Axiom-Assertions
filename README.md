# Axiom

Axiom is an open-source .NET assertion library focused on high performance, deterministic failure output, and practical extensibility.

## Status

This repository is currently being built in milestones.

## Planned Packages

- `Axiom.Core`
- `Axiom.Assertions`
- `Axiom.Analyzers` (scaffold)
- `Axiom.Vectors` (scaffold)

## Build

```bash
dotnet restore Axiom.sln
dotnet build Axiom.sln -c Release
dotnet test Axiom.sln -c Release
```
